### 1. What is the difference between null and undefined?
- undefined হলো কোনো একটা ভেরিয়েবল তৈরি হয়েছে কিন্তু এখনো ভ্যালু দেওয়া হয়নি, পরবর্তীতে ভ্যালু দেওয়া হবে। এছাড়াও undefined হলো JavaScript এর দেওয়া default খালি একটা মান। আর null নিজেই একটা ভ্যালু তবে এইটা হলো প্রোগ্রামার ইচ্ছা করেই একটা ভেরিয়েবল এর ভ্যালু খালি রাখতে চায়
